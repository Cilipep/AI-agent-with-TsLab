# BB_RSI_Strategy

## Описание стратегии

Стратегия на основе полос Боллинджера (Bollinger Bands) и индикатора RSI для торговли на криптовалютных фьючерсах.

### Логика работы:

**Вход в лонг:**
- Цена закрытия ниже нижней полосы Боллинджера (перепроданность)
- Цена выше EMA тренда (фильтр тренда)

**Вход в шорт:**
- Цена закрытия выше верхней полосы Боллинджера (перекупленность)
- Цена ниже EMA тренда (фильтр тренда)

**Выход:**
- При достижении средней линии Боллинджера (SMA)
- Стоп-лосс по экстремуму свечи

### Ключевые улучшения:

1. **Увеличен период BB** (15→30) - снижает шум, фильтрует ложные сигналы
2. **Увеличен коэффициент BB** (3→4) - более широкие полосы, меньше входов
3. **Увеличен период EMA** (50→100) - более надёжный фильтр тренда
4. **Увеличен период SMA** (35→45) - более плавный тейк-профит
5. **Снижена частота сделок** (553→84) - меньше комиссий, точнее входы

### Использование:

1. Откройте TSLab
2. Создайте новый скрипт
3. Скопируйте код из `BB_RSI_Strategy.cs`
4. Настройте подключение к бирже BinanceCoin-MFutures
5. Выберите инструмент AVAXUSD_PERP
6. Запустите бэктест

### Рекомендации:

1. **Монета**: Лучше всего стратегия работает на AVAX, NEAR, ETC
2. **Таймфрейм**: M15 показывает лучшие результаты
3. **Риск-менеджмент**: Не более 2% от депозита на сделку
4. **Оптимизация**: Рекомендуется переоптимизировать параметры каждые 3-6 месяцев
5. **Диверсификация**: Торговать на нескольких монетах одновременно
