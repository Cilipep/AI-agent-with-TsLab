# GridBotRSI

Грид-стратегия с RSI фильтрацией для торговли криптовалютами.

## Быстрый старт

1. Откройте TSLab
2. Импортируйте стратегию из папки `GridBotRSI`
3. Установите инструмент: ADAUSD_PERP
4. Запустите бэктест

## Параметры по умолчанию

- Шаг сетки: 0.5%
- Стоп-лосс: 2%
- Тейк-профит шорт: 1.5%
- RSI фильтр: 35/50

## Результаты

Лучший результат: +400.90% за 60 дней на ADAUSD_PERP

## Файлы

- `Описание.docx` - подробное описание стратегии
- `parameters.txt` - параметры стратегии
- `config.json` - конфигурация в JSON формате
- `README.md` - этот файл
