# Стратегия GridBotRSI

## Описание стратегии

**Название:** GridBotRSI  
**Тип:** Грид-стратегия с RSI фильтрацией  
**Инструмент:** DOGEUSD_PERP (BinanceCoin-MFutures)  
**Таймфрейм:** 30 минут  
**Начальный депозит:** 25 USDT  

## Логика работы

### Вход в лонг:
- Условие: RSI_30 < 40 (перепроданность)
- Цена входа: Close + GridConst (текущая цена + шаг сетки)
- Размер позиции: 12.5 DOGE (50% от баланса)

### Выход из лонга:
- Тейк-профит: Close + GridConst (шаг сетки)
- Стоп-лосс: Close - StopStep (стоп-шаг)

### Вход в шорт:
- Условие: RSI_30 > 60 (перекупленность)
- Цена входа: Close + GridConst
- Размер позиции: 12.5 DOGE

### Выход из шорта:
- Тейк-профит: Close - ShortTakeProfit
- Стоп-лосс: Close + StopStep

## Параметры стратегии

| Параметр | Значение | Описание |
|----------|----------|----------|
| GridConst | 0.005 (0.5%) | Шаг сетки для входа/выхода |
| StopStep | 0.02 (2%) | Стоп-лосс |
| ShortTakeProfit | 0.02 (2%) | Тейк-профит для шортов |
| RSI_Oversold | 40 | Порог перепроданности |
| RSI_Overbought | 60 | Порог перекупленности |
| Shares | 12.5 | Размер позиции (DOGE) |
| Commission | 0.05% | Комиссия за сделку |

## Индикаторы

- **RSI_1** - RSI с периодом 1
- **RSI_5** - RSI с периодом 5
- **RSI_15** - RSI с периодом 15
- **RSI_30** - RSI с периодом 30 (основной фильтр)
