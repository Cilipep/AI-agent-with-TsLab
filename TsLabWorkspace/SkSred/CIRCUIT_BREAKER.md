# Circuit Breaker для SkSred

## Описание

Circuit Breaker (аварийный выключатель) — это механизм защиты от чрезмерных потерь. При превышении заданного процента просадки стратегия автоматически закрывает все позиции и прекращает торговлю.

## Добавленные блоки

### 1. Equity (Текущий капитал)
- **Формула**: `Position + Cash`
- **Описание**: Рассчитывает текущий капитал = стоимость позиций + свободные средства

### 2. PeakEquity (Пиковый капитал)
- **Формула**: `Highest(Equity, 1000)`
- **Описание**: Отслеживает максимальный достигнутый капитал за последние 1000 баров

### 3. DrawdownPct (Просадка в %)
- **Формула**: `IIF(PeakEquity > 0, (PeakEquity - Equity) / PeakEquity * 100, 0)`
- **Описание**: Рассчитывает текущую просадку в процентах от пика

### 4. CircuitBreaker (Условие срабатывания)
- **Формула**: `DrawdownPct >= MaxDrawdownPct`
- **Описание**: Срабатывает при превышении максимальной просадки

### 5. CloseOnBreaker (Закрытие позиций)
- **Тип**: ClosePositionByMarketItem
- **Описание**: Закрывает все позиции по рынку при срабатывании circuit breaker

## Параметры

| Параметр | Значение по умолчанию | Диапазон | Описание |
|----------|----------------------|----------|----------|
| Max Drawdown, % | 20 | 10-50 | Максимальная просадка для срабатывания |

## Настройка в TSLab

1. Откройте скрипт `SkSred.tscript` в TSLab
2. Перейдите в настройки параметров (Parameters)
3. Найдите параметр **"Max Drawdown, %"** (CircuitBreaker_MaxDrawdownPct)
4. Установите желаемое значение:
   - **10-15%** — консервативный (раннее срабатывание)
   - **20%** — умеренный (по умолчанию)
   - **25-30%** — агрессивный (позднее срабатывание)
   - **40-50%** — очень агрессивный

## Графики

Добавлен новый график **pane_drawdown** (красная линия), отображающий:
- Текущую просадку в процентах
- Уровень порога circuit breaker

## Логика работы

```
┌─────────────────────────────────────────────────────────┐
│                    КАЖДЫЙ БАР                            │
├─────────────────────────────────────────────────────────┤
│  1. Рассчитать Equity = Position + Cash                 │
│  2. Обновить PeakEquity = Highest(Equity, 1000)         │
│  3. Рассчитать DrawdownPct = (Peak - Current) / Peak    │
│  4. Если DrawdownPct >= MaxDrawdownPct:                 │
│     → Закрыть все позиции (CloseOnBreaker)              │
│  5. Иначе:                                              │
│     → Продолжить торговлю по стратегии SMA              │
└─────────────────────────────────────────────────────────┘
```

## Рекомендации

### Для разных стилей торговли:

| Стиль | Max Drawdown | Описание |
|-------|--------------|----------|
| Скальпинг | 5-10% | Быстрое закрытие при малейших losses |
| Дейтрейдинг | 10-15% | Умеренная защита |
| Свинг-трейдинг | 15-25% | Средняя защита |
| Позиционная торговля | 25-40% | Допускает большие колебания |

### Оптимизация параметра:

1. **Бэктест**: Протестируйте разные значения MaxDrawdownPct
2. **Анализ**: Сравните количество срабатываний и итоговую прибыль
3. **Баланс**: Слишком низкое значение → частые ложные срабатывания
4. **Баланс**: Слишком высокое значение → запоздалая защита

## Примеры использования

### Консервативный (рекомендуется для начинающих)
```
Max Drawdown = 15%
```
- Частые закрытия при небольших просадках
- Защита капитала выше среднего

### Умеренный (по умолчанию)
```
Max Drawdown = 20%
```
- Сбалансированная защита
- Подходит для большинства стратегий

### Агрессивный
```
Max Drawdown = 30%
```
- Допускает значительные просадки
- Подходит для волатильных рынков

## Восстановление после срабатывания

После срабатывания circuit breaker стратегия **автоматически прекращает торговлю**. Для возобновления:

1. **Ручной способ**: Перезапустите скрипт в TSLab
2. **Автоматический**: Добавьте логику восстановления (требует доработки)

## Дополнительные настройки

### Изменение периода для PeakEquity
Текущая настройка: `Highest(Equity, 1000)` — 1000 баров

Для более короткого периода (быстрее реагирует):
```
Highest(Equity, 200)  // 200 баров
```

Для более длинного периода (медленнее реагирует):
```
Highest(Equity, 2000)  // 2000 баров
```
