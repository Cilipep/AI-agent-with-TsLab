# SkSred - Стратегия на скользящих средних

## Описание стратегии

Торговая система на основе пересечения двух скользящих средних (Golden Cross / Death Cross).

## Правила торговли

### Основные правила

| Правило | Описание |
|---------|----------|
| **Покупка (Long)** | Краткосрочная SMA пересекает долгосрочную SMA **снизу вверх** |
| **Продажа (Short)** | Краткосрочная SMA пересекает долгосрочную SMA **сверху вниз** |
| **Удержание** | Оставаться в позиции до появления противоположного сигнала |

### Определения

- **C(t)** — цена закрытия дня t (сегодня t=1)
- **MA1** — период краткосрочной скользящей средней (по умолчанию: 5)
- **MA2** — период долгосрочной скользящей средней (по умолчанию: 20)

### Формулы

```
MA1 = [C(t) + C(t+1) + ... + C(t+MA1_period-1)] / MA1_period
MA2 = [C(t) + C(t+1) + ... + C(t+MA2_period-1)] / MA2_period
```

### Параметры

| Параметр | Значение | Описание |
|----------|----------|----------|
| MA1_period | 5 | Период краткосрочной SMA |
| MA2_period | 20 | Период долгосрочной SMA |
| Commission | 0.05% | Комиссия за сделку |

### Ограничения

- **MA2_period ≥ 2 × MA1_period** (правило из документа)

## Логика входа/выхода

### Вход в Long (Покупка)

```
Условие: MA1(t) > MA2(t) И MA1(t-1) < MA2(t-1)
Действие: Купить по рыночной цене
```

### Выход из Long (Продажа)

```
Условие: MA1(t) < MA2(t) И MA1(t-1) > MA2(t-1)
Действие: Продать по рыночной цене
```

### Удержание позиции

```
Если в Long и MA1(t) > MA2(t) → Не делать ничего
Если в Short и MA1(t) < MA2(t) → Не делать ничего
```

## Код стратегии (Advanced Trader Script)

```
Period_1 = 5
MA1_Today = sma[c:, Period_1, 0, 0]
MA1_Yesterday = sma[c:, Period_1, 0, 1]

Period_2 = 20
MA2_Today = sma[c:, Period_2, 0, 0]
MA2_Yesterday = sma[c:, Period_2, 0, 1]

// Вход в Long
if MA1_Today > MA2_Today then
    if MA1_Yesterday < MA2_Yesterday then
        Order #1
        buy net_position + 1 at MOO
    endif
endif

// Выход из Long (Вход в Short)
if MA1_Today < MA2_Today then
    if MA1_Yesterday > MA2_Yesterday then
        Order #2
        sell net_position + 1 at MOO
    endif
endif
```

## Рекомендации по оптимизации

1. **Увеличить периоды SMA** — текущие 5/20 слишком чувствительны к шуму
2. **Добавить фильтр тренда** — использовать дополнительный индикатор для подтверждения
3. **Добавить Stop Loss / Take Profit** — ограничить убытки
4. **Тестировать на разных таймфреймах** — H4, D1 могут дать лучшие результаты
