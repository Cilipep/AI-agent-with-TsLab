Нейросетевые модели

LSTMModel

Рекуррентная сеть
Input → LSTM(64,2) → FC(32) → ReLU → FC(1)

Запоминает последовательности. Hidden: 64, Layers: 2, Dropout: 0.35

TCNModel

Temporal Convolutional Network
Input → Conv1d(k=3,d=2^i) × N → BN → FC三层

Каузальные свёртки с дилатацией. Моделирует длинные зависимости.

TransformerModel

Self-Attention
Input → Proj → PosEnc → TransEnc(h=8) → FC三层

Multi-head attention. Pre-norm, GELU. 4x feed-forward.

AttentionTransformer

Custom Attention Blocks
Input → Proj → PosEnc → N×AttnBlock → FC

Кастомные блоки с residual connections.

MultiTaskModel

Multi-Task Learning
LSTM → ClassHead + RegHead + ConfHead

3 выхода: buy/sell, изменение цены, уверенность.

Ансамбли

Ensemble

Weighted Average
logits = Σ(w_i × model_i(x))

N моделей, веса оптимизируются на validation.

StackingEnsemble

Meta-Learning
Base → Concat → Meta(n→16→1)

Base заморожены, meta обучается на предсказаниях.
