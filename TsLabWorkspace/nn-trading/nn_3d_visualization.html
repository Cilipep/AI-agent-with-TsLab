Нейросетевые модели

LSTMModel

Рекуррентная сеть
Input → LSTM(hidden, layers) → FC(32) → ReLU → Dropout → FC(1)

Запоминает последовательности. Берёт последний timestep для предсказания.

TCNModel

Temporal Convolutional Network
Input → Conv1d(kernel=3, dilation=2^i) × N → BN → ReLU → FC三层

Каузальные свёртки с дилатацией. Эффективно моделирует длинные зависимости.

TransformerModel

Self-Attention
Input → Projection → PosEnc → TransformerEncoder(heads=8) → FC三层

Multi-head self-attention. Pre-norm, GELU активация. 4x feed-forward.

AttentionTransformer

Custom Attention Blocks
Input → Projection → PosEnc → N × AttnBlock → FC

Кастомные attention блоки с residual connections. Лучше для коротких последовательностей.

MultiTaskModel

Multi-Task Learning
LSTM → Classification Head + Regression Head + Confidence Head

3 выхода: buy/sell (классификация), изменение цены (регрессия), уверенность (0-1).

LSTMRegression / TCNRegression / TransformerRegression

Регрессионные варианты
Base Model → FC(64) → FC(32) → FC(1)

Предсказывают величину изменения цены (не направление).

Ансамбли и sklearn модели

Ensemble

Weighted Average
logits = Σ(weight_i × model_i(x))

N базовых моделей, веса оптимизируются на validation.

StackingEnsemble

Meta-Learning
Base Models → Concat → Meta FC(n_base → 16 → 1)

Base модели заморожены, meta-learner обучается на их предсказаниях.

HybridEnsemble

Neural + sklearn
3 NN + 4 sklearn → Weighted Average

Комбинация LSTM/Transformer/Attention + XGBoost/CatBoost/LightGBM/RF.

SklearnModelWrapper

Обёртка для sklearn
flatten(seq) → predict_proba → logit transform

Преобразует sklearn предсказания в logits для совместимости с Ensemble.

Пайплайн обработки данных

1. Скачивание данных

Binance API: 5m, 15m, 30m, 1h, 4h, 1d, 1w, 1M. Автоматическая пагинация и кеширование.

2. Мультитаймфреймовые фичи

Для каждого ТФ: EMA(10,20,50), RSI(14), MACD, BB(20), ATR(14), ADX(14), Stoch, Williams, CCI, ROC, Momentum. Forward-fill к дневным барам.

3. Auto-Select фичей

Random Forest importance + Mutual Information. Автоматический отбор топ-40-50 из 50-170 фичей.

4. Обучение моделей

Walk-forward: anchored expanding window. Train 85/15 split. Early stopping по val loss. 3 seed для ансамбля.

5. Ансамблирование

HybridEnsemble: 3 NN + 4 sklearn. Веса через grid search. Confidence threshold 0.5+.

6. Backtest

Trailing stop, dynamic sizing по ATR, transaction costs (0.1% + 0.05% slippage + 0.05% spread).
