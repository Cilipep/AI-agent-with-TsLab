# Spread Trading Bot for Bybit

Автоматическая стратегия торговли на разнице цен между коррелированными перпетуэлами.

## Установка

```bash
cd spread-trading-bot
pip install -r requirements.txt
```

## Тестирование

```bash
python test_bot.py
```

## Запуск

```bash
python spotread_bot.py
```

## Конфигурация

Редактируйте `config.py` для настройки параметров:

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| PAIR | Инструменты для торговли | BTCUSDT/ETHUSDT |
| ENTRY_Z | Порог входа (Z-Score) | 2.0 |
| EXIT_Z | Порог выхода (Z-Score) | 0.5 |
| STOP_Z | Стоп-лосс (Z-Score) | 3.0 |
| POSITION_SIZE_USDT | Размер позиции в USDT | 5 |
| LEVERAGE | Кредитное плечо | 10 |

## Логика стратегии

1. **Мониторинг**: Рассчитывает Z-Score спреда каждую минуту
2. **Вход**: Когда Z-Score > 2.0 или < -2.0
3. **Выход**: Когда Z-Score возвращается к 0.5 или -0.5
4. **Стоп-лосс**: Когда Z-Score > 3.0 или < -3.0

## Управление рисками

- Максимальный размер позиции: 5 USDT на ногу
- Кредитное плечо: 10x
- Автоматический стоп-лосс

## Логи

Все операции записываются в `spread_bot.log`
